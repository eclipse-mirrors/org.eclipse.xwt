Commands

Identifier:
org.eclipse.xwt.pde.commands

Since:
[Enter the first release in which this extension point appears.]

Description:
This extension point is used to register a command with a name

Configuration Markup:

<!ELEMENT extension (command*)>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED>


<!ELEMENT command EMPTY>

<!ATTLIST command

name  CDATA #REQUIRED

class CDATA #REQUIRED>

  • name - the name
  • class - The class of command must implement the interface org.eclipse.xwt.ICommand

Examples:
Here is an example to define a commands: 

   <extension
         point="org.eclipse.xwt.pde.commands">
      <command
            class="org.eclipse.xwt.demo.MessageCommand"
            name="Message">
      </command>
   </extension>
